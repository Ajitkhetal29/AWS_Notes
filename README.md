# AWS_Notes

DevOps is a continuous process of planning, coding, building, testing, integrating, deploying, operating, and monitoring applications using automation tools and cloud platforms like AWS.

Goal -> 
1. Build fast
2, Deploy Easily
3. fixx issues quickly
4. automate everything

Phases ->
1. Plan 
2. Code
3. Build  tools -> maven , graddle
4. Test -> Selenium, Cucumber
5. Integrate (CI) - Merge new code with existing code automatically , jenkins
6. Cd - Deply app to server -> 
    Docker (containerization)
    Kubernetes (container management)
    Ansible (automation)
7. Operate 
8. Monitor

What is CI?
Automatically merge + build + test code whenever developer pushes changes

🔄 Flow:
1. Developer writes code
2. Push to Git
3. CI tool (like Jenkins) triggers
4. Code is built
5. Tests run automatically

🛠️ Tools:
Jenkins
GitHub Actions
GitLab CI

What is CD?
Automatically deploy application to server after successful CI
What is CD?

Automatically deploy application to server after successful CI

CI ensures code is continuously tested and integrated, while CD ensures that validated code is automatically deployed to production
